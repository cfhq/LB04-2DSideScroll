# Oak Woods 

A simple 2D side-scrolling platformer set in an enchanted forest.

## About the Game

Oak Woods is a short platforming adventure where players journey through a magical woodland. Navigate platforms, avoid obstacles, and make your way through the forest as you explore its enchanting atmosphere.

## Features

* 2D side-scrolling platformer gameplay
* Woodland-themed environment
* Platforming challenges and obstacles
* Simple and accessible controls

## Controls

| Action               | Key             |
| -------------------- | --------------- |
| Move Left/backwards  | A / Left Arrow  |
| Move Right/forward   | D / Right Arrow |
| Jump                 | Space           |

## How to Play

Reach the end of each level by overcoming obstacles and carefully timing your jumps. Stay alert and use your platforming skills to navigate through the enchanted woods.

* Unity
* C#
